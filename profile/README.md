## How it works

Every installed app can become a network node.
Your message travels through real users' phones before reaching its destination.
No phone number. No email. No identity.

## Architecture

- **Core Server** - central control, encrypted storage
- **Mobile Contributors** - users who opt-in to relay encrypted packets
- **VPS Nodes** - dedicated relays run by operators worldwide

## Privacy guarantees

- End-to-end encrypted - server never sees message content
- Anonymous accounts - no phone number, no email
- Ghost Mode - messages that leave no trace
- Burn After Reading - cryptographic proof of destruction

## Node software

The node relay software is open source.
Anyone can verify it never touches message content.

→ [inkogn/node](https://github.com/inkognapp/node)
